## Short Description (1200 characters max)

Developed a comprehensive Islamic mobile application featuring accurate Qibla direction finder, automated prayer time notifications with Azan sound, complete Quran with translations, Hadith collections, Ramadan tracker, Zakat calculator, and premium subscription system. Built with Flutter for cross-platform deployment (Android & iOS) and integrated with Firebase for authentication and cloud storage.

### Project Architecture

#### Clean Architecture Implementation

```
lib/
├── main.dart (App entry point)
├── bindings/ (Dependency injection)
├── controllers/ (Business logic)
├── models/ (Data models)
├── services/ (API & external services)
├── views/ (UI screens)
├── widgets/ (Reusable components)
├── routes/ (Navigation)
├── core/ (Config, theme, utils)
└── utils/ (Helper functions)
```
